Resumen ejecutivo

Por qué migrar ahora

Hoy el proceso vive en un archivo Excel con macros que cada usuario instala y sincroniza. Migrar a una plataforma web lo convierte en un sistema gobernado, auditable y escalable.

El momento es ahora: cuanto más crece el archivo, más frágil, lento y difícil de auditar se vuelve el modelo actual.
